HTML

- Crie um arquivo (index.html) na pasta do projeto
- Estrutura básica do HTML, com head e body:

<html>
<head> (Cabeçalho)
  <title></title> (Título)
  <meta name="description" content="">
</head>

- Cabeçalho da página

<body> (Corpo)
  <header>
    <a href=""> (Link)
      <img src="" alt="PetLife"> (Imagem)
    </a>
    <nav>
      <a href="#"></a>
      <a href="#"></a>
      <a href="#"></a>
      <a href="#"></a>
    </nav>
  </header>
</body>
</html>

- Conteúdo da página

<main>
  Seções
    <section id=""> (identificador)
      <img src="" alt="" width="" height="">
      <div>
        <h1></h1>
        <p></p>
        <a href="" target="_blank"></a>
        <a href="#"></a>
      </div>
      <img src="" alt="">
    </section> (Seção)
</main>

- Listas

<ul>
<li></li>
</ul>

- Loading="lazy" (Carregamento atrasado)

- <footer> (Rodapé)

- mailto (email)

- link -> rel -> stylesheet
